# react-redux-sample-app
Author: Patryk Smul

## Description
Simple app allowing to lookup for random users generated from specified external or local API.

## App launch instructions:
1. `npm i` or `npm install` in main folder
2. `npm run webapp-with-server` - command starting webapp and local fake server as concurrent tasks

If you want to launch webapp and server as separate tasks/jobs, please follow those commands:
1. `npm run start` - starting local webapp on default (3000) port
2. `npm run json-server` - starting local fake json-server on 3005 port

## Other available instructions:
1. `npm run test` - command for printing unit tests results for defined specs
2. `npm run build` - build production ready application
